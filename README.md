# AI English Tutor - Agentic AI System

## 📌 Overview

This project implements an Agentic AI system for English learning, acting as a virtual tutor.

## 🧠 Features

- Q&A Agent (grammar & vocabulary explanation)
- Exercise Agent (practice generation)
- Grading Agent (writing correction)
- Orchestrator Agent (intent routing)

## 🏗️ Architecture

- Multi-agent system with central orchestrator
- Prompt-based agent design
- LLM inference via API (Groq)

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Open-source LLM APIs

## 🚀 How to run

```bash
pip install -r requirements.txt
python -m streamlit run app/streamlit_app.py
```
